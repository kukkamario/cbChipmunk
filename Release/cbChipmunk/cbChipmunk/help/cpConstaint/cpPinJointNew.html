cpPinJointNew


   KUVAUS

Luo uuden liitoksen kahden rungon välille. PinJointissa ankkuripisteiden väli pysyy jatkuvasti vakiona. Vakio väli määräytyy luonti hetken etäisyyden mukaan.
   KÄYTTÖ
cpPinJointNew(body1,body2,p1x#,p1y#,p2x#,p2y#)

  • body1 = Ensimmäinen runko.

  • body2 = Toinen runko.

  • p1x# = Ensimmäisessä rungossa kiinni olevan ankkuripisteen runkoon suhteessa oleva x-koordinaatti.

  • p1y# = Ensimmäisessä rungossa kiinni olevan ankkuripisteen runkoon suhteessa oleva y-koordinaatti.

  • p2x# = Toisessa rungossa kiinni olevan ankkuripisteen runkoon suhteessa oleva x-koordinaatti.

  • p2y# = Toisessa rungossa kiinni olevan ankkuripisteen runkoon suhteessa oleva y-koordinaatti.
